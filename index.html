Scripts FiveM - Detalhes

Sobre o projeto

Este projeto consiste em um script avançado para servidores FiveM (modificação do GTA V) que implementa um sistema de drogas com diversas funcionalidades:

  • Sistema de coleta de ingredientes em locais específicos do mapa
  • Processamento e fabricação de substâncias
  • Sistema de inventário personalizado para armazenamento
  • Mecânica de venda para NPCs em diferentes locais
  • Sistema de risco policial com níveis de procura

Tecnologias utilizadas

  • Lua (linguagem principal)
  • Framework QBcore para FiveM
  • MySQL para armazenamento de dados
  • JavaScript para elementos de interface
Lua

Exemplo de código


-- Sistema de coleta de recursos
RegisterNetEvent('harvest:collect')
AddEventHandler('harvest:collect', function(itemName, count)
    local xPlayer = ESX.GetPlayerFromId(source)
    
    -- Verificar se o jogador tem espaço no inventário
    if xPlayer.canCarryItem(itemName, count) then
        -- Adicionar item ao inventário
        xPlayer.addInventoryItem(itemName, count)
        
        -- Notificar o jogador
        TriggerClientEvent('esx:showNotification', source, 
          'Você coletou ' .. count .. 'x ' .. ESX.GetItemLabel(itemName))
    else
        -- Inventário cheio
        TriggerClientEvent('esx:showNotification', source, 
          'Seu inventário está cheio!')
    end
end)
